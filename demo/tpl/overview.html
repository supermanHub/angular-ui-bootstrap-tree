Angular UI Bootstrap Tree

uibTree is a angular directive used to create a treeview in html. It has a very high performance and allow to create unlimited nest treeview. It also provider a util service which used to operate the tree view.

How to use
  • Step 1: Include the directive and its css 
    <script src="path_to/ui-bootstrap-tree.js"></script>
<link rel="stylesheet" href="path_to/ui-bootstrap-tree.css" type="text/css" />
  • Step 2: Create you angular app and reject the directive 
    var yourApp = angular.module('yourApp', [
	'ui.bootstrap', //based on bootstrap, please inlcude angular ui bootstrap in html first
	'ui.bootstrap.tree'
])
  • Step 3: Create a controller 
    yourApp.controller('TreeCtrl', ['$scope', function($scope){
	$scope.treeModel = {
		label: 'Root'
	};

	//Or
	//$scope.treeModel = [{
	//	label: 'Root 1',
	//	children: [
	//		{
	//			label: 'Child 1'
	//		},
	//		{
	//			label: 'Child 2'
	//		}
	//	]
	//},{
	//	label: 'Root 2',
	//	children: [
	//		{
	//			label: 'Child 1'
	//		},
	//		{
	//			label: 'Child 2'
	//		}
	//	]
	//}]
}]);
  • Step 4: Use the directive in HTML 
    <uib-tree tree-model="treeModel"></uib-tree>
  • Step 5: If you want to use the uibTreeUtil,please inject it in controller 
    yourApp.controller('TreeCtrl', ['$scope', 'uibTreeUtil', function($scope,uibTreeUtil){
	$scope.treeModel = {
		label: 'Root'
	};

	$scope.expandAll =  function(){
		uibTreeUtil.expandAll($scope.treeModel);
	}

}]);
uibTreeUtil Functions
  • uibTreeUtil.expandAll(treeModel)
  • uibTreeUtil.expandByLevel(level) level is number large than 0
  • uibTreeUtil.collapseAll(treeModel)
  • uibTreeUtil.getSelectedTreeModel(treeModel)
